# CMDU_DDE_DOCK
深度 Linux 系统网速任务栏插件，鼠标悬浮显示开机时间、CPU使用率、内存使用率、下载字节、上传字节。  

![alt](preview.png)  
内存占用大于90%时显示为红色以示警告，注意减少内存预防死机。  
![alt](preview90.png)  

安装：终端 ./install.sh  
卸载：终端 ./uninstall.sh
